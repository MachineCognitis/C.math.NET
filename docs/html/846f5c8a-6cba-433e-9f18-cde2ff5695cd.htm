Welcome to the C.math.NET Library

The C.math.NET library implements several C Standard functions that are missing from the .NET framework. Functions include among others: frexp(Double, Int32), ilogb(Double), ldexp(Double, Int32), logb(Double), scalbn(Double, Int32), copysign(Double, Double), nextafter(Double, Double), isunordered(Double, Double), and fpclassify(Double). Both double and single precision functions are implemented. All functions are static and their names follow the C Standard.

All functions have been thoroughly tested using a large set of unit tests that cover all limit cases to guarantee that all functions behave as specified by the C standard.
