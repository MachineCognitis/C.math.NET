Remarks

IEC 60559 recommends that fromNumber be returned whenever fromNumber == towardNumber. These functions return towardNumber instead, which makes the behavior around zero consistent: nexttoward(Single, Single)(-0.0, +0.0) returns +0.0 and nexttoward(Single, Single)(+0.0, -0.0) returns –0.0.

The nexttoward(Single, Single) function is equivalent to the nextafter(Single, Single) function.

See nexttoward for the C standard documentation.
