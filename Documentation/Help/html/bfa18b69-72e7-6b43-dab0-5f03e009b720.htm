A fraction in the range [0.5, 1) so that number = fraction * 2^exponent.
Remarks

Special values are treated as follows.

  • If number is ±0, it is returned, and 0 is returned in exponent.
  • If number is infinite, it is returned, and an undefined value is returned in exponent.
  • If number is NaN, it is returned, and an undefined value is returned in exponent.

The function frexp(Single, Int32), together with its dual, ldexp(Single, Int32), can be used to manipulate the representation of a floating-point number without direct bit manipulations.

The relation of frexp(Single, Int32) to logb(Single) and scalbn(Single, Int32) is:

exponent = (number == 0) ? 0 : (int)(1 + logb(Single)(number))
fraction = scalbn(Single, Int32)(number, -exponent)

See frexp for the C standard documentation.
